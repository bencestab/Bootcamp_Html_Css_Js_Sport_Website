Változók Definícióm: A változókban adatokat (string, number...) tudunk tárolni azért,hogy az adott értéket többször fel tudjuk használni. Elég elővenni a változót ahelyett, hogy mindig megadnánk az értéket. Példa: a = 1234, b = 567 a + b (1234 + 567) a - b (1234 - 567) c = "Sziasztok" d = "Lányok" f = "Fiúk" c + d = "Sziasztok Lányok" c + f = "Sziasztok Fiúk" Youtube videó: https://www.youtube.com/watch?v=Dc3cXhlMAO8 Cikk: https://webiskola.hu/javascript-ismeretek/javascript-valtozok-es-konstansok-var-let-es-const-magyarul/ ...Mik azok a változók?... Deklaráció Definícióm: A változók létrehozását, bevezetését deklarációnak nevezzük. 2015 óta 3 lehetséges módja van ennek: -var : 2015 előtt csak ezzel a kulcsszóval lehetett változót létrehozni. -let : ezzel a kulcsszóval olyan változót hozhatunk létre, ami csak egy blokk hatókörében érvényes. -const : a const (konstans) kulcsszóval olyan változót tudunk létrehozni, amelyet a későbbiek folyamán nem lehet felülírni. Olyan értéket deklarálunk const változóban, mint például a hónapok neveit, amik minden évben ugyanabban a sorrendben követik egymást és sosem változik a nevük. Youtube videó: https://www.youtube.com/watch?v=9WIJQDvt4Us Cikk: https://webiskola.hu/javascript-ismeretek/javascript-valtozok-es-konstansok-var-let-es-const-magyarul/ ...a var,let, const története...
